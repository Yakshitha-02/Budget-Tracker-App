# Budget-Tracker-App

A personal finance management application designed to help users monitor expenses and manage budgets efficiently.

## Features

- Expense tracking
- Budget planning
- Dashboard analytics
- Data visualization
- Responsive UI

## Tech Stack

- JavaScript
- React
- Vite
- HTML
- CSS

## Functionalities

- Add expenses
- Categorize transactions
- View analytics
- Monthly budget monitoring
